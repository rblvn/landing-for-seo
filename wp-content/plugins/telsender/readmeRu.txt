TelSender v0.9.6 - плагин что работает с contact form 7 и магазином woocomerce в wordprees.Отправляет заявки из форм в чат телеграмма. Без прокси

== Description ==

Contact form 7 заработал славу как бесплатного конструктора форм с гибкой системой настроек.

Я же предлагаю это только улучшить.А именно отправлять письма не только на почту, а и в чат телеграм.
Так же есть возможность отправлять заказы из вашего wooccommerce магазина. Пример сообщение на скриншоте №3.


== Features ==

* Отправляет заявки из contact-form 7 в телеграм;
* Отправлять заказы из wooccommerce в телеграм;
* Быстро доходят сообщения;
* Удобно в использовании;
* Эффективно;


== Installation ==
1- Установить и активировать
2- В настройках ввести токен и id вашего чата.
3- в формах ниже указать какие хотите получать в телеграм.
